# Lead Pipeline

> **How to use:** Update this document weekly (or after any significant outreach). Track both practitioner leads (supply side) and client leads (demand side) separately. Move leads through stages as they progress. Flag any leads that involve clinical partnerships or institutional agreements for human review before committing.

---

## Pipeline Stages

```
Awareness → Interested → Qualified → Onboarding → Active → Churned
```

### Practitioner Onboarding Checklist

For each practitioner moving from **Qualified → Onboarding**:

- [ ] Registration number provided (AHPRA / AASW / other)
- [ ] Registration verified against public register ⚠️ **Human review required**
- [ ] Scope of practice reviewed and confirmed ⚠️ **Human review required**
- [ ] ABN verified (via ABN Lookup tool in-app)
- [ ] Halaxy integration connected (if applicable)
- [ ] Profile approved and made discoverable
- [ ] Welcome email sent
